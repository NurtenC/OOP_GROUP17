Detailed Description

Manages and tracks budget amounts, handling expenses and incomes.

The Budget class provides functionalities to add expenses and incomes, set and retrieve the current budget, and represent the budget as a formatted string. It ensures that the budget is not exceeded by alerting the user when necessary.
