Detailed Description

Manages the start screen interface, including welcome messages and navigation buttons.

The StartScreenGroup class inherits from Fl_Group and provides UI components for users to navigate to the Login or Register screens. It includes a welcome label and buttons for initiating login or registration processes within the Expense Tracker application.
